# Accessibility Statement

## Our Commitment

We are committed to ensuring digital accessibility for all users of the Physical AI & Humanoid Robotics textbook. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Measures We Take

To ensure accessibility, we:
- Use semantic HTML elements appropriately
- Provide alternative text for images
- Ensure sufficient color contrast
- Maintain keyboard navigation functionality
- Implement proper heading structure
- Use ARIA attributes where needed
- Test with screen readers
- Follow WCAG 2.1 AA guidelines

## Technical Specifications

The textbook is built using:
- Docusaurus framework (compliant with HTML5 standards)
- React components with proper accessibility attributes
- Responsive design that works across devices
- Proper landmark navigation

## Conformance Status

The textbook is designed to meet:
- W3C Web Content Accessibility Guidelines (WCAG) 2.1 AA level
- Section 508 of the Rehabilitation Act
- European EN 301 549 accessibility standard

## Feedback

If you experience accessibility issues or need content in an alternative format, please contact us at accessibility@example.com.
